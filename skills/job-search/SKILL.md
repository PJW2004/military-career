# 채용공고 검색 스킬

## 개요

이 스킬은 잡코리아(JobKorea)와 사람인(Saramin)에서 채용공고를 검색하는 MCP 도구를 활용합니다.

## 사용 가능한 도구

### search_jobs : 단일 회사 채용공고 검색
- `company_name` (필수): 검색할 회사명. **사용자가 입력한 그대로** 전달
- `platform` (선택): "jobkorea", "saramin", "all" (기본값: "all")
- `page` (선택): 페이지 번호 (기본값: 1)

### search_jobs_bulk : 여러 회사 일괄 검색
- `company_names` (필수): 회사명 배열. **사용자가 입력한 그대로** 전달
- `platform` (선택): "jobkorea", "saramin", "all" (기본값: "all")
- 내부적으로 10개씩 병렬 요청하여 빠르게 검색
- 결과는 임시 파일로 저장되며, 요약과 파일 경로를 반환

## 검색 결과 형식

각 채용공고는 다음 정보를 포함합니다:
- 플랫폼 (잡코리아/사람인)
- 회사명
- 공고 제목
- 경력 조건 (신입/경력/무관)
- 학력 조건
- 근무 지역
- 마감일
- 공고 URL

## 사용 가이드

1. 회사명은 정확하게 입력할수록 좋습니다
2. 검색 결과가 없으면:
   - 회사 정식명/약어를 바꿔서 시도해보세요 (예: "삼전" → "삼성전자")
   - 플랫폼을 지정하여 개별 검색해보세요
3. 다수 회사 검색 시 `search_jobs_bulk`가 훨씬 빠릅니다
4. 병역특례 업체 검색이 필요하면 `/military` 커맨드를 안내하세요
