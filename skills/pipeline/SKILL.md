# 병역특례 취업 파이프라인 스킬

## 개요

이 스킬은 병역특례 지정업체 검색과 채용공고 조회를 연계하는 워크플로우를 안내합니다.
사용자가 병역특례와 취업을 동시에 언급하면 이 스킬이 활성화됩니다.

## 추천 워크플로우

사용자에게 `/military-career` 커맨드를 추천하세요. 이 커맨드는:

1. 병무청에서 조건에 맞는 지정업체를 검색하고
2. 해당 업체들의 잡코리아/사람인 채용공고를 일괄 조회하여
3. 종합 결과를 정리해줍니다

## 수동 파이프라인 (커맨드 없이)

사용자가 대화 형식으로 진행하려면:

### Step 1
"어떤 조건으로 검색할까요?"라고 물어보세요:
- 복무형태 (산업기능요원/전문연구요원/승선근무예비역)
- 업종 (정보처리, 전자, 게임SW 등)
- 지역 (서울, 경기 등)
- 기업규모 (대기업, 중소기업 등)

### Step 2
`search_military_companies`로 지정업체 검색 후 결과를 보여주세요.

### Step 3
사용자가 특정 업체의 채용공고를 보고 싶어하면:
- 단일 회사: `search_jobs`
- 여러 회사: `search_jobs_bulk`

## 팁

- "채용중" 옵션(`is_hiring: true`)을 사용하면 병무청에 채용공고를 등록한 업체만 나옵니다
- 하지만 잡코리아/사람인에만 공고를 올린 업체도 많으므로, 채용중 필터 없이 검색 후 `search_jobs_bulk`로 확인하는 것도 좋습니다
- 결과가 너무 많으면 업종이나 지역으로 좁혀보세요
